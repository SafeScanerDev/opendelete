💀 Terminal System v1.0 — Access Control & Security

Terminal System v1.0 — это кастомная консольная оболочка с уникальным визуальным стилем (Skull & Arrow ASCII) и встроенной системой авторизации. Предназначена для управления модулями защиты и мониторинга подписки.
🔑 Система лицензирования / Licensing
В скрипт встроена проверка кодов доступа, которая определяет длительность сессии пользователя:
Код (Key)	Срок (Duration)	Тип доступа
N9sa-	30 дней	Standard
Nopa233-	90 дней	Advanced
3333	Навсегда	Lifetime / Admin
🕹️ Функционал меню
После успешного ввода кода открывается доступ к следующим инструментам:
⚔️ Фатальное уничтожение: Мгновенный запуск протоколов очистки системы от подозрительных процессов и вредоносного кода.
🔍 Сканер: Глубокий анализ файлов на наличие сигнатур стилеров и майнеров.
📅 Время подписки: Проверка текущего статуса лицензии и оставшегося времени работы.
🎨 Визуальное оформление
Skull Design: Уникальный ASCII-арт черепа со стрелой при входе.
Dynamic Colors: Система меняет цвет консоли в зависимости от этапа (красный при входе, зеленый в меню, синий при сканировании).
Adaptive Text: Текст подписки автоматически подтягивается в зависимости от введенного ключа.
🚀 Запуск
Откройте файл Terminal_System.bat.
Введите один из валидных кодов доступа.
Используйте цифровые клавиши (1-4) для навигации по меню.
